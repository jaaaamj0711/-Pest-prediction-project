# 도심 속 돌발해충 발생 요인 분석 및 예측 모델 개발

## 💁분석 개요

여름철 자주 등장하는 돌해충으로 인해 주민들이 일상생활에 많은 피해을 입고 있다.

시기나 장소에 한정되지 않고 돌발적으로 발생하여 피해 발생 예측이 힘들다.

### ✨ 아이디어

**따라서 지역별 기상, 산림, 건물, 가로등 주요 요인을 분석하여 우선 방제 지역을 선정하고자 한다.**
